# URL Shortener

A simple and lightweight URL shortener built with **HTML**, **Tailwind CSS**, and **JavaScript** using the **is.gd API**.
Clean UI, fast performance, and fully frontend-based.

---

## ✨ Features

* Shorten long URLs instantly
* Clean and responsive UI
* Copy shortened links easily
* Powered by the is.gd API
* No backend required
* Built with vanilla JavaScript + Tailwind CSS

---

## 🛠️ Tech Stack

* HTML
* Tailwind CSS
* JavaScript
* is.gd API

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/InshRonin/URLMonk.git
```

Open the project folder:

```bash
cd URLMonk
```

Run locally by opening `index.html` in your browser.

## 💡 How It Works

1. User enters a long URL
2. JavaScript sends a request to the is.gd API
3. API returns a shortened URL
4. The shortened link is displayed instantly
